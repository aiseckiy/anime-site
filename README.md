# DANGO

Anime catalog prototype uploaded from Codex.

## Features

- Black + pink UI
- Main page with recommendations, history, and continue watching
- Full anime catalog
- Jikan API previews and metadata
- Title page with seasons and episode grid
- Player page with saved progress in localStorage
- Local profile modal with email/password mock registration and banner/avatar upload

Open `index.html` directly or enable GitHub Pages for this repository.
